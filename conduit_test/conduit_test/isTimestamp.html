Validates that a value is a ISO8601 timestamp.

Usage:

    var response = await client.request("/foo").get();

Matcher isTimestamp = predicate((String str) {
  try {
    DateTime.parse(str);
    return true;
  } catch (e) {
    return false;
  }
}, "is timestamp");
